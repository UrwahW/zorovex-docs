## Goal

Allow organization owners to read and update basic organization settings.

## Acceptance Criteria

- [ ] `GET /organizations/:id` returns org name, slug, and settings
- [ ] `PATCH /organizations/:id` updates name and settings (owner/admin only)
- [ ] Requests scoped to user's organization (no cross-tenant access)
- [ ] Non-members receive 403
- [ ] Endpoints documented in `03_Backend/ORGANIZATIONS.md` and `03_Backend/API.md`
- [ ] Tests cover read, update, and cross-tenant denial
