# Objective

Implement the Receptionist configuration module.

This module allows business owners to configure how the AI Receptionist behaves without writing prompts.

Business owners configure business behaviour.

The platform generates AI behaviour from these settings.

No LLM integration is implemented in this task.

# Responsibilities

Implement Receptionist Configuration.

Allow editing of:

- Personality
- Tone
- Greeting
- Goals
- Knowledge Sources
- Escalation Rules

Configuration is stored in the AI Agent configuration JSON.

---

# Configuration Structure

```json
{
  "personality": {
    "style": "professional",
    "friendliness": "medium",
    "emoji_usage": false
  },
  "greeting": {
    "enabled": true,
    "message": ""
  },
  "tone": {
    "verbosity": "balanced",
    "language": "en"
  },
  "goals": {
    "book_appointments": true,
    "generate_leads": true,
    "promote_packages": false,
    "answer_questions": true
  },
  "knowledge_sources": [
    "business_profile",
    "services",
    "staff",
    "business_brain"
  ],
  "escalations": {
    "refund_requests": true,
    "complaints": true,
    "payment_issues": true,
    "low_confidence": true
  }
}
```
