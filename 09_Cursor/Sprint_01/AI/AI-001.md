# API Behaviour

GET /api/v1/ai/agents

Returns all agents for current organization.

GET /api/v1/ai/agents/:id

Returns single agent.

Returns 404 if not found.

Returns 403 if agent belongs to another organization.

POST /api/v1/ai/agents

Creates new agent.

Slug generated from name if not provided.

Slug must be unique per organization.

PATCH /api/v1/ai/agents/:id

Partial update allowed.

DELETE /api/v1/ai/agents/:id

Soft delete not required.

Receptionist agent cannot be deleted.

Returns 422 if deletion blocked.
