# Objective

Implement the Zorovex AI Runtime.

This module becomes the execution engine for every AI Agent.

The Runtime is responsible for:

- Loading AI Agent configuration
- Loading organization context
- Building execution context
- Loading business knowledge
- Selecting goals
- Selecting tools
- Executing the AI provider
- Returning structured results

The Runtime must remain provider-independent.

OpenAI, Anthropic, Gemini and future providers are adapters.
