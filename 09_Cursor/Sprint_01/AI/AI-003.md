# Objective

Implement the AI Playground.

The Playground is a safe environment where business owners can test their AI Receptionist before enabling it for customers.

No messages are sent to real communication channels.

No LLM provider is connected.

No runtime is executed.

The Playground validates the complete AI experience using a simulated runtime.

Allow business owners to:

- Inspect AI behaviour
- Understand business rules
- Verify configuration

The Playground is for testing.

Not deployment.

# Mock Runtime

The runtime should return deterministic responses.

No randomness.

No AI provider.

No API keys.

Use the Receptionist configuration to influence the simulated output.
