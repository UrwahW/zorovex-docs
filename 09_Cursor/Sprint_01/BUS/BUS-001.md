## Goal

Implement a multi-step onboarding wizard so new owners can configure their business profile after registration.

## Acceptance Criteria

- [ ] Wizard shown after first login when business profile is incomplete
- [ ] Steps: business name, type, timezone, contact info (minimum)
- [ ] `POST /business-setup` saves profile scoped to organization
- [ ] `GET /business-setup` returns current profile and completion status
- [ ] Completing wizard redirects to dashboard
- [ ] Progress persists if user leaves and returns
- [ ] Backend and UI documented in `03_Backend/BUSINESS_SETUP.md` and `04_Frontend/BUSINESS_SETUP_UI.md`
