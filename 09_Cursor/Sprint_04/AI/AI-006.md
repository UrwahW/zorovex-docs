# Objective

Implement the Omnichannel Conversation Platform.

This module becomes the single entry point for all customer conversations regardless of communication channel.

Supported channels

- WhatsApp
- Instagram
- Facebook Messenger
- Website Chat
- Twilio Voice

All conversations must follow the same pipeline.

---

# Philosophy

Every communication provider is treated as a channel.

Business logic must never depend on the provider.

Once a message enters the platform it becomes a normalized conversation event.

The AI Runtime and Workflow Engine should not know whether the message originated from WhatsApp, Instagram, Facebook, Website Chat or Voice.

Resume thread.

Otherwise

Create new thread.
