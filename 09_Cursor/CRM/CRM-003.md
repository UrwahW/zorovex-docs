# Duplicate Detection

Detect duplicates by

- Email
- Phone
- WhatsApp Number

Display duplicate suggestions.

Allow merge only after user confirmation.

---

# Contact Merge

Allow merging two contacts.

Rules

- Preserve booking history
- Preserve notes
- Preserve documents
- Preserve communication history
- Preserve timeline
- Preserve tags

Merged contact becomes archived.

Merge operation must be reversible only through database restore.

---

# Segments

Support dynamic segments.

Examples

- VIP Customers
- New Leads
- No Bookings in 90 Days
- Birthday This Month
- High Value Customers
- Recently Contacted

Segments are rule-based.

No manual membership.

---

# Saved Filters

Allow users to save CRM searches.

Examples

- Today's Leads
- VIP Clients
- Bridal Customers

Users may set one default filter.

---

# Import

Support CSV Import.

Required mapping

- First Name
- Last Name
- Phone
- Email
- WhatsApp
- Lead Source

Preview import before execution.

Skip duplicates if selected.

---

# Export

Support CSV Export.

Allow exporting

- Current Filter
- Current Segment
- Entire CRM

---

# Bulk Actions

Support

- Archive
- Delete
- Change Status
- Change Lead Source
- Add Tags
- Remove Tags

Bulk operations must be transactional.
