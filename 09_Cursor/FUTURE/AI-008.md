# Objective

Implement the AI Memory Engine.

The Memory Engine enables Zorovex AI to remember structured information across conversations while remaining deterministic and auditable.

Memory is intended to improve customer experience without allowing the AI to invent or modify facts.

---

# Scope

Implement

- Customer Memory
- Business Memory
- Workflow Memory
- AI Memory
- Memory Ranking
- Memory Retrieval
- Memory Expiration
- Memory API

---

# Customer Memory

Store

- Preferred language
- Preferred staff
- Preferred services
- Favourite products
- Preferred appointment times
- Communication preference
- Known allergies
- Special notes

---

# Business Memory

Store

- Busy periods
- Frequently booked services
- Preferred booking patterns
- Common customer questions
- Business preferences

---

# Workflow Memory

Store

- Previous bookings
- Payment history
- Cancellation history
- Reschedule history
- Workflow outcomes

---

# AI Memory

Store

- Previous interactions
- Successful recommendations
- Previous follow-ups
- Customer preferences confirmed by conversations

---

# Memory Rules

Memory must

- Be structured
- Be auditable
- Be editable
- Be deletable
- Never contain hallucinated information

---

# Memory Retrieval

Runtime should retrieve

- Relevant customer memory
- Relevant workflow memory
- Relevant business memory

based on current conversation.
