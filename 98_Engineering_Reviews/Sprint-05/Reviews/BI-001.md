**Task:** BI-001 (Business Intelligence Platform)

---

## Summary

Implemented the centralized Business Intelligence platform. The `Bi` namespace aggregates operational data across CRM, bookings, operations, communications, AI runtime, and workflows into KPIs, trends, business health scoring, deterministic insights, and rule-based recommendations. An executive dashboard is exposed at `/analytics` with CSV export support.

No module-specific reporting logic was duplicated; BI queries raw operational models directly per spec.

---

## Services Added

| Service | Purpose |
|---------|---------|
| `Bi::BusinessIntelligenceService` | Facade for all BI endpoints |
| `Bi::DashboardService` | Executive dashboard composition + caching |
| `Bi::KpiEngine` | Centralized KPI calculation with trends/thresholds |
| `Bi::KpiRegistry` | KPI definitions by category |
| `Bi::MetricsAggregator` | Domain analytics orchestration |
| `Bi::TrendAnalyzer` | Period-over-period historical trends |
| `Bi::BusinessHealthService` | 0–100 health score with factor weighting |
| `Bi::InsightEngine` | Data-driven business insights |
| `Bi::RecommendationEngine` | Deterministic rule-based recommendations |
| `Bi::CacheService` | Hourly/daily/monthly `Rails.cache` tiers |
| `Bi::ExportService` | CSV export flattening |
| `Bi::EventLogger` | Structured BI audit logging |
| `Bi::Analytics::*` | Customer, booking, revenue, staff, AI, comms, operations analytics |

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/v1/bi/dashboard` | Executive dashboard payload |
| GET | `/api/v1/bi/overview` | Business overview summary |
| GET | `/api/v1/bi/kpis` | KPI list with trends |
| GET | `/api/v1/bi/customers` | Customer analytics |
| GET | `/api/v1/bi/bookings` | Booking analytics |
| GET | `/api/v1/bi/revenue` | Estimated revenue analytics |
| GET | `/api/v1/bi/staff` | Staff performance analytics |
| GET | `/api/v1/bi/operations` | Capacity/utilization analytics |
| GET | `/api/v1/bi/communications` | Channel/conversation analytics |
| GET | `/api/v1/bi/ai` | AI performance analytics |
| GET | `/api/v1/bi/health` | Business health score |
| GET | `/api/v1/bi/insights` | Generated insights |
| GET | `/api/v1/bi/export` | CSV export (`report`, `period` params) |

Query params: `period` (`today|week|month|quarter|year`), `refresh` (boolean).
